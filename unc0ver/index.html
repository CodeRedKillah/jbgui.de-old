unc0ver or unc0ver dark?


If you're looking to jailbreak with unc0ver, you have two options; the original unc0ver by Pwn20wnd, or unc0ver dark, a fork of the unc0ver project developed by Shady and Diatrus, which adds support for Sileo, and allows you to change the look of your jailbreak application using a theme switcher, with preset options (True Black, Dark Purple, and White).


Keep in mind, unc0ver (original) does not offer Sileo, or any application theming options. On unc0ver Dark, you have the ability to use Cydia and Sileo at the same time.



Installation (with computer)


This is the preferred method of installing jailbreak apps onto one's device, but there is a method to do it without a computer if needed. If you have a computer, then read on.


To get started, you'll need to install Cydia Impactor on your computer. Simply click the link and install it like any other program.
